Networking Basics

What is Networking?

Networking is the practice of connecting computers and other devices to share resources and information.

Basics Of Networking

Network Types:

1. LAN (Local Area Network): Covers a small area, like a home or office.

2. WAN (Wide Area Network): Covers a large geographical area, such as the internet.

3. MAN (Metropolitan Area Network): Spans a city or a large campus.

Components:

1. Nodes are Devices such as computers, printers, and routers connected in a network.

2. Routers Direct data packets between networks.

3. A Switch Connects devices within a single network, forwarding data to specific devices.

4. A Modem Converts digital data to analog for transmission over phone lines and vice versa.



Common Protocols

HTTP/HTTPS:

Web page transfer protocols.

FTP:

File transfer protocol.

SMTP/POP3/IMAP:

Email protocols.

SSH

Secure Shell for secure network services over an unsecured network.



Communication Models

OSI Model: Seven-layer framework for network communication:

Physical: Hardware transmission of raw data.

Data Link: Node-to-node data transfer.

Network: Routing of data packets.

Transport: Reliable data transfer.

Session: Managing sessions between applications.

Presentation: Data translation, encryption, and compression.

Application: Network services to applications.



TCP/IP Model: Four-layer framework:

Network Interface: Physical and data link layers.

Internet: Network layer.

Transport: Transport layer

Application: Application layer.



Key Concepts

Bandwidth:

Maximum data transfer rate of a network.

Latency:

Delay in data transmission.

Firewall:

Security system that controls incoming and outgoing network traffic.

VPN (Virtual Private Network):

Secure connection over a public network.
