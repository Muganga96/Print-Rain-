Development & Design Basics

What is Front End Development?

This is the practice of converting data to a graphical interface for users to view and interact with through digital interaction using HTML, CSS, and JavaScript.

Core Technologies

HTML (HyperText Markup Language):

This is Structure of web pages & Elements include headings, paragraphs, links, images, and more.

CSS (Cascading Style Sheets):

Running the code to ensure it works as expected.

JavaScript:

This refers to Interactivity and dynamic content. It Manipulates HTML and CSS. Libraries and frameworks like React, Angular, and Vue.js.



CKey Concepts

DOM (Document Object Model):

This represents the page structure. JavaScript interacts with the DOM to update content dynamically.

Responsive Design:

This Ensures web pages render well on various devices and window sizes. Techniques include fluid grids, flexible images, and media queries.

Accessibility:

This involves Designing and developing websites usable by people with disabilities. Follows standards such as WCAG (Web Content Accessibility Guidelines).



UI/UX Design Principles

User Interface (UI):

This mainly Focuses on the look and feel of the application. It consists of Consistency, simplicity, and feedback.

User Experience (UX):

This mainly Focuses on the overall experience and usability. It consists of User research, wireframing, prototyping, and usability testing.



Design Tools

Wireframing and Prototyping:

Tools like Adobe XD, Sketch, Figma for creating design mockups and interactive prototypes.

Graphic Design:

Tools like Adobe Photoshop and Illustrator for creating visual elements.
