Programming Basics

What is Programming?

Programming is the process of designing and building executable computer software to accomplish a specific task.

It enables the automation of tasks, solving of problems, and creation of applications and systems that perform complex operations.



Basics Of Programming

Languages: Programming is done using various languages such as Python, Java, C++, JavaScript, etc. Each language has its own syntax and use cases.

Syntax: The set of rules that defines the combinations of symbols considered to be correctly structured programs in a language.

Code: Instructions written by a programmer in a specific programming language.



Development Process

Writing Code:

Using a text editor or an Integrated Development Environment (IDE).

Testing:

Running the code to ensure it works as expected.

Debugging.

Identifying and fixing errors in the code.

Compiling/Interpreting:

Converting code into machine-readable format. Compilers translate the entire program before execution, whereas interpreters translate code line-by-line during execution.

Version Control:

Using systems like Git to manage changes to the codebase.



Common Paradigms

Procedural Programming:

Based on procedure calls; includes languages like C.

Object-Oriented Programming (OOP):

Based on the concept of "objects"; includes languages like Java and Python.

Functional Programming::

Treats computation as the evaluation of mathematical functions; includes languages like Haskell and Lisp.
